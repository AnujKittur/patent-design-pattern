# Patent Database Resources for Construction Robotics Research

## Primary Patent Databases

### 1. United States Patent and Trademark Office (USPTO)

**Main Patent Search:**
- **URL**: https://patft.uspto.gov/
- **Full-Text Search**: https://patft.uspto.gov/netahtml/PTO/search-bool.html
- **Advanced Search**: https://patft.uspto.gov/netahtml/PTO/search-adv.htm
- **Description**: Official US patent database with full-text search capabilities. Contains all US patents and patent applications.

**USPTO Patent Application Database (AppFT):**
- **URL**: https://appft.uspto.gov/netahtml/PTO/search-bool.html
- **Description**: Contains published patent applications (not yet granted)

**USPTO Bulk Data Downloads:**
- **URL**: https://bulkdata.uspto.gov/
- **Description**: Download patent data in bulk formats (XML, JSON) for research purposes
- **Patent Grant Data**: https://bulkdata.uspto.gov/data/patent/grant/redbook/bibliographic/
- **Patent Application Data**: https://bulkdata.uspto.gov/data/patent/application/redbook/bibliographic/

**USPTO Patent Public Search:**
- **URL**: https://ppubs.uspto.gov/pubwebapp/
- **Description**: Modern interface for searching patents and applications

### 2. European Patent Office (EPO)

**Espacenet - Free Patent Search:**
- **URL**: https://worldwide.espacenet.com/
- **Description**: Free access to over 130 million patent documents worldwide
- **Advanced Search**: https://worldwide.espacenet.com/advancedSearch?locale=en_EP

**European Patent Register:**
- **URL**: https://register.epo.org/
- **Description**: Search European patent applications and granted patents

**EPO Patent Information Services:**
- **URL**: https://www.epo.org/searching-for-patents.html
- **Description**: Comprehensive patent search tools and databases

**EPO Bulk Data:**
- **URL**: https://www.epo.org/searching-for-patents/data/bulk-data-sets.html
- **Description**: Bulk downloads of European patent data

### 3. World Intellectual Property Organization (WIPO)

**PATENTSCOPE:**
- **URL**: https://patentscope.wipo.int/search/en/search.jsf
- **Description**: Free access to international patent applications (PCT) and national patent collections
- **Advanced Search**: https://patentscope.wipo.int/search/en/advancedSearch.jsf

**WIPO Global Brand Database:**
- **URL**: https://www3.wipo.int/branddb/en/
- **Description**: Search for trademarks and industrial designs

**WIPO IP Statistics:**
- **URL**: https://www.wipo.int/ipstats/en/
- **Description**: Patent statistics and analytics

### 4. Google Patents

**Google Patents Search:**
- **URL**: https://patents.google.com/
- **Description**: Comprehensive patent search engine covering multiple patent offices
- **Advanced Search**: https://patents.google.com/advanced
- **Features**: Full-text search, citation analysis, patent families, PDF downloads

**Google Patents Dataset:**
- **URL**: https://console.cloud.google.com/marketplace/product/google_patents_public_datasets
- **Description**: Google Cloud dataset for bulk patent data analysis

### 5. IEEE Xplore

**IEEE Xplore Digital Library:**
- **URL**: https://ieeexplore.ieee.org/
- **Description**: Technical papers, standards, and patents related to engineering and technology
- **Patent Search**: Use search filters to find patents in IEEE database

## Recommended Search Strategies

### Key Patent Classification Codes for Construction Robotics:

1. **B25J** - Manipulators; Chambers Provided with Manipulation Devices
   - B25J9/00 - Programme-controlled manipulators
   - B25J15/00 - Gripping heads

2. **E04G** - Scaffolding; Forms; Shuttering
   - E04G21/00 - Preparing, conveying, or working-up building materials
   - E04G23/00 - Repairing, securing, or restoring buildings

3. **E04B** - General Building Constructions
   - E04B1/00 - Constructions in general

4. **B66C** - Cranes; Load-Engaging Elements or Devices for Cranes, Capstans, Winches, or Tackles
   - B66C13/00 - Other constructional features or details

   - G05B19/00 - Programme-control systems

## Data Access Methods

### 1. Web Scraping (with proper permissions):
- Use APIs where available
- Respect robots.txt and terms of service
- Consider rate limiting

## Important Notes

1. **Legal Considerations:**
   - Patents are public documents, but respect copyright on database interfaces
   - Check terms of service for each database
   - Some databases require registration for advanced features

2. **Data Format:**
   - Patents are typically available in PDF, XML, or HTML formats
   - Bulk data often comes in XML format
   - Consider data cleaning and normalization needs

3. **Research Ethics:**
   - Cite sources appropriately
   - Respect database usage policies
   - Consider rate limiting for automated access
